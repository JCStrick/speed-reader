# Speed Reader

A speed reading application using RSVP (Rapid Serial Visual Presentation) with ORP (Optimal Recognition Point) highlighting to enable high-speed reading with minimal eye movement.

## Features

- Word-by-word display with ORP letter highlighting
- Paste text or upload PDF/TXT/DOCX files
- Adjustable reading speed (200-1000+ WPM)
- Auto mode with gradual speed increase
- Play/pause, rewind, fast-forward controls
- Full-screen distraction-free mode
- Document library with progress tracking
- Dark theme optimized for reading

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- pdfjs-dist for PDF extraction
- mammoth for DOCX support
- localStorage for persistence
